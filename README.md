# How to test

```
Run all
$ ./vhls-check -f -v

Run csim only
$ ./vhls-check -f -v --csim-only

Run hls only
$ ./vhls-check -f -v --hls-only

Run cosim only
$ ./vhls-check -f -v --cosim-only
```
